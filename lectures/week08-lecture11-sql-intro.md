# Introduction to SQL


Phyical model of a database describes:
- specific implementation of the model, offers final database abstraction


Basic Commands are:

- Create
- Drop
- Alter
- Insert
- Select
- Update
- Delete


Common Data Types
- Int
- Varchar(n)
- Decimal (p,s)  -> p total digits, s digits after decimal
- Date (yyyy-mm-dd)


Operations and Functions

3 types:

- Logic, ex. and, or, not
- Relational, ex. <, >, =, <=, >=, <>, like, in, Between
- Aggregation, ex. sum, avg, count, max, min


To manipulate database structure you use
- Create
- Drop
- Alter
